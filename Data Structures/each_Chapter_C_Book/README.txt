you can get whole directory from 00-zip-files
so first click 00-zip-files

Then..  Suppose you are in a directory called X in your local computer

first get booklib.zip and save it under X

~/X> unzip booklib.zip
~/X> cd booklib
~/X/booklib> make
~/X/booklib> cd ..
~/X> 


get any other zip file say 08-Abstract-Data-Types.zip
and save it under X

~/X> unzip 08-Abstract-Data-Types.zip 
~/X> cd 08-Abstract-Data-Types
~/X/08-Abstract-Data-Types> make
~/X/08-Abstract-Data-Types> 

now you can edit source files, make, and run programs
